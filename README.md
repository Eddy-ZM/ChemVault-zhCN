# ChemVault 中文站

这是 ChemVault 的中文静态站版本，用于中文内容展示和 ICP 备案审核准备。

## 本目录包含

- 中文化首页、主要功能页、关于页、团队页、开发者页
- `pages/filing.html`：网站说明与备案信息页
- 原站静态资源、数据文件和前端脚本
- Cloudflare Pages 构建脚本与部署配置

## 本地构建

```bash
npm run build
```

构建产物会输出到 `dist/`。

## 备案号替换

当前页脚和 `pages/filing.html` 中的 ICP 备案号为“待填写”。备案号取得后，请全局替换：

- `ICP备案号：待填写`
- 如有公安备案号，也可在页脚和备案信息页补充。

## 说明

化学记录中的英文名称、缩写和外部数据库名称部分保留英文，以保持学术检索准确性。本站内容仅用于学习、科研资料整理和公开信息检索，不构成实验、安全、医疗或商业建议。
